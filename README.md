# HR Analytics Dashboard

## Project Overview
This project is an interactive HR Analytics Dashboard developed using Power BI. It helps analyze employee data to understand attrition trends, workforce distribution, employee demographics, and departmental performance. The dashboard provides insights that support HR decision-making and employee retention strategies.

---

## Objectives
- Analyze employee attrition.
- Identify departments with high attrition.
- Understand employee distribution by age and gender.
- Analyze salary slabs and experience levels.
- Support HR with data-driven insights.

---

## Tools Used
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization

---

## Dashboard Features
- Total Employees
- Active Employees
- Attrition Count
- Attrition Rate
- Average Age
- Average Experience
- Department-wise Attrition
- Salary Slab Analysis
- Age Group Distribution
- Gender-wise Attrition
- Experience Analysis
- Department-wise Employee Count

---

## Key Insights
- Overall Attrition Rate: **16.6%**
- Total Employees: **1,405**
- Active Employees: **1,172**
- Total Attrition: **233**
- Highest attrition observed in Operations and Administration departments.
- Employees with 0–3 LPA salary showed higher attrition.
- Most employees belong to the 26–35 age group.
